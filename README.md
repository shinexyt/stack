# 叠叠翻车

基于 Matter.js 的物理叠塔小游戏。当前为**阶段一**：自动横移的物品可点击落下、发生物理堆叠、统计层数，并在已放置物品掉出画面时结束游戏。

## 本地运行

```bash
npm install
npm run dev
```

构建静态产物：

```bash
npm run build
```

## 操作

- 点击屏幕或按空格：让当前物品落下。
- 左右方向键：在物品落下前微调其位置。
- 游戏结束后点击“再试一次”：重新开始。

## 添加物品与调参

阶段一使用纯色方块作为占位物品。游戏参数统一在 `src/config.js`，可调整重力、物品尺寸、自动移动速度、摩擦力、落定阈值与掉出边界等。后续接入 PNG 美术时，可在 `src/game.js` 的物品创建逻辑中根据物品类型添加对应纹理。
