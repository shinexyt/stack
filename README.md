# 叠叠翻车

基于 Matter.js 的物理叠塔小游戏。当前已完成**阶段二**：自动横移的物品可点击落下、发生物理堆叠、统计层数，并在已放置物品掉出画面时结束游戏；塔升高时镜头平滑跟随，并提供落地和翻车的声画反馈。

## 本地运行

```bash
npm install
npm run dev
```

构建静态产物：

```bash
npm run build
```

## 操作

- 点击屏幕或按空格：让当前物品落下。
- 左右方向键：在物品落下前微调其位置。
- 游戏结束后点击“再试一次”：重新开始。

## 添加物品与调参

游戏参数统一在 `src/config.js`，可调整重力、物品尺寸、自动移动速度、摩擦力、镜头和反馈效果等。6 个透明背景物品资源位于 `public/assets/items/`，物品清单位于 `src/items.js`；添加新物品时，在清单中登记资源路径和占位色即可。
